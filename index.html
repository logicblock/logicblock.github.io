Welcome to Logic Blocks project

Logic Blocks is an open hardware project to make a small logic gates into breakout boards.

Sometimes when you try to do a logic circuit on a breadboard you go crazy to identify the pinout of each logic gate, so I made these small boards with logic gates symbols on a top silkscreen of the board =)

Available Logic Blocks:

7408 - Quad 2-input AND gates

Working on:

  • 7432 - Quad 2-input OR gates
  • 7404 - Hex Inverter - Hexa NOT gates
  • 7400 - Quad 2-input NAND gate
  • 7402 - Quad 2-input NOR gate
  • and more...
